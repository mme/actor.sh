How it works

01
Create

Tell your agent to spin up actors for each task. Each gets its own isolated branch and workspace — no collisions, no conflicts.

"spin up three actors to fix auth, update docs, and add tests"
02
Run

Actors work in parallel in the background. Check progress anytime without interrupting them — or just let them finish.

"what are my actors doing?"
03
Ship

When an actor finishes, merge it or open a pull request — all without leaving your conversation.

"merge the auth actor and open a PR"

Works with Claude Code, Codex, Cursor, and more.
